# [alaska-cache-memcache](https://github.com/maichong/alaska-cache-memcache)
Alaska memcache driver

## configure

```javascript

cache: {
  type: 'alaska-cache-memcache',
  host: 'localhost',
  port: 11211,
  // Default maximum age in milliseconds, 0 for forever
  maxAge: 3600 * 1000
}

```
